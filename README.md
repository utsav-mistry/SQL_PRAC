Lightweight SQL sandbox web app for safely practicing PostgreSQL queries.  
Frontend is React, backend is Node.js/Express, and the sandbox database runs in a Postgres instance (Render in the original deployment).

## Features
- Interactive SQL editor with run / run-selection support.
- Sidebar showing schema hierarchy and table column previews.
- Sandbox reset to restore initial sample data.
- Simple auth (admin / practice users) for demo purposes.
- Info modal showing static server placeholders (host and versions).
